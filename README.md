## 🧠 About Me

Desenvolvedor com experiência em **sistemas que não podem falhar**.

Trabalho profissionalmente com **Delphi** desde 2022 — mantendo e evoluindo ERPs corporativos críticos com integração fiscal (NF-e), bancos relacionais e arquiteturas legadas. O tipo de sistema onde um bug tem consequência real.

Paralelamente, construí do zero o **Leite Fácil**: um SaaS completo em produção real, com multi-tenancy, app mobile (Android/iOS), CI/CD e cliente ativo — o que me deu fluência real em **Node.js**, **TypeScript**, **React** e **PostgreSQL**, não só em tutoriais.

---

## 🚀 Featured Project

**[Leite Fácil](https://github.com/GuilhermeProvenci)** — SaaS em produção real

> Sistema completo para gestão de produção leiteira. Monorepo full stack com arquitetura multi-tenant, banco isolado por fazenda, app mobile nativo (Android/iOS) e CI/CD automatizado.

| Camada | Tecnologia |
|--------|-----------|
| Frontend | React · TypeScript · Tailwind CSS |
| Mobile | Capacitor (Android/iOS) |
| Backend | Node.js · Express · TypeScript |
| Banco | PostgreSQL · Drizzle ORM |
| DevOps | Docker · GitHub Actions · CI/CD |
| Testes | Vitest · Playwright |
| Extras | Firebase (push) · JWT · WebSocket |
